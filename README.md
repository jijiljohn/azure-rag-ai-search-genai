# Azure RAG Application using Azure AI Search & Azure OpenAI

A production-ready **Retrieval-Augmented Generation (RAG)** application built using:

- **Azure OpenAI**
- **Azure AI Search (Vector Search)**
- **FastAPI**
- **Python**

This project demonstrates how to ingest documents, generate embeddings, perform vector search, and generate grounded AI responses using retrieved context.
